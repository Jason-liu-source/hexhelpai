# HexHelp AI — Free Developer Tools

🌐 **Live site:** [https://hexhelpai.com](https://hexhelpai.com)

A collection of free, fast, privacy-first online tools for developers and everyday users. No signup required. No tracking. Everything runs in your browser.

## 🔧 Tools

### 🌐 Network Tools
| Tool | URL | Description |
|------|-----|-------------|
| IP Address Inspector | [/tools/ip.html](https://hexhelpai.com/tools/ip.html) | Public IPv4/IPv6, geolocation, user agent |
| DNS Lookup | [/tools/dns.html](https://hexhelpai.com/tools/dns.html) | A, AAAA, MX, TXT, CNAME, NS, SOA records |
| WHOIS / Domain Info | [/tools/whois.html](https://hexhelpai.com/tools/whois.html) | Registration, registrar, nameservers |

## 🚀 Stack
- Pure HTML / CSS / JavaScript — zero frameworks, zero dependencies
- Served via **Nginx** on Amazon Linux 2023
- **Let's Encrypt** SSL (auto-renewing)
- All tools run 100% client-side (nothing sent to server)
